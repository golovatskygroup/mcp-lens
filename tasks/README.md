# Tasks

Эта директория — бэклог инженерных задач по улучшению `mcp-lens` (mcp-proxy).

Принципы:
- `query` остаётся единственным публичным tool (см. `internal/server/server.go`).
- Улучшения делаем максимально **общими**, чтобы они работали не только для Grafana, но и для GitHub/Jira/Confluence/любых upstream tools.
- Любые внешние интеграции в тестах — только opt-in (build tags) и с `t.Skip` без env.

Формат задач:
- Один файл = одна задача с понятными acceptance criteria и планом тестирования.
- Нумерация `0001-...` для стабильных ссылок.
